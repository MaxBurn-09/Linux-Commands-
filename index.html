Basic File Commands

CommandDescription

pwdPrint current working directory
cdChange directory

mkdirCreate new directory
rmdirRemove empty directory

rm -rRecursively delete directory and contents
rm -fForce delete files
cpCopy files or directories
mvMove or rename files
touchCreate an empty file
catDisplay file content
tacDisplay file content in reverse
headView first lines of a file
tailView last lines of a file
tail -fFollow file output (log files)
statView detailed file info
fileIdentify file type
findLocate files/directories
locateSearch files using database
updatedbUpdate file database for locate
basenameExtract filename from path
dirnameExtract directory from path
readlinkResolve symbolic links
realpathPrint the absolute path
